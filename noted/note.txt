A. Endpoint Eksternal (Public/Customer & Partner Facing)

1. Registrasi Konsumen Baru -> POST /api/v1/customers/register
2. Melihat Profil Sendiri -> GET /api/v1/me/profile
3. Mengupdate Profil Sendiri -> PUT /api/v1/me/profile
4*. Melihat Limit Peminjaman Sendiri -> GET /api/v1/me/limits
5*. Melihat Riwayat Transaksi Sendiri -> GET /api/v1/me/transactions

6*. Mengecek Ketersediaan Limit Konsumen -> POST /api/v1/partners/check-limit
7*. Membuat Transaksi Baru -> POST /api/v1/partners/transactions

B. Endpoint Internal (Admin Facing)

1. Mendapatkan Daftar Konsumen (untuk Verifikasi, dll) -> GET /api/v1/admin/customers
2. Melihat Detail Satu Konsumen -> GET /api/v1/admin/customers/{customerId}
3. Memverifikasi atau Menolak Konsumen -> POST /api/v1/admin/customers/{customerId}/verify
4*. Menetapkan atau Mengupdate Limit Konsumen -> POST /api/v1/admin/customers/{customerId}/limits
